# Zenodo Publishing Guide

## Recommended Record Type

- Upload type: Publication
- Publication type: Preprint
- Access: Open
- Licence: Creative Commons Attribution 4.0 International
- Version: 1.0
- Publication date: June 25, 2026

## Files to Upload

Upload the ZIP package:

```text
Study_Hours_Marks_Prediction_Zenodo_Package_v1.0.zip
```

Set this file as the main preview:

```text
Study_Hours_Marks_Prediction_IEEE_Preprint_v1.0.pdf
```

## Missing Optional Author Details

The following details were not added because verified values were not provided:

- ORCID
- Public email address
- College or university affiliation
- Public GitHub repository URL

These details can be added to the Zenodo metadata before publication.

## DOI Step

If a DOI is reserved before publication:

1. Add the DOI to the paper.
2. Rebuild the PDF and DOCX.
3. Replace the files in the draft.
4. Check the Zenodo preview again.
5. Publish only after the final check.

Do not type a made-up DOI.
